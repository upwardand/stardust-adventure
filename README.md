# 星尘大冒险 Stardust Adventure

一个单文件 HTML5 横版平台闯关游戏——零依赖，电脑、手机、平板打开网页就能玩。

**🎮 在线试玩：** https://upwardand.github.io/stardust-adventure/

## 玩法

- **10 个超长主题关卡**：绿茵草原 → 黄昏遗迹 → 熔岩堡垒 → 星空之巅 → 落日矿洞 → 云端天路 → 冰封峡谷 → 幽暗墓穴 → 风暴要塞 → 魔王王座，每关 300 格以上的超长流程、2~5 个检查点，屏幕顶部有关卡进度条
- **选关地图与三星挑战**：通关自动解锁下一关并存档（关掉浏览器也不丢），每关三颗星——通关 / 宝石全收集 / 目标时间内通关，还会记录每关最快纪录
- **四场 Boss 战**：熔岩史莱姆王（第三关）、星芒史莱姆王（第四关）、冰霜史莱姆王（第七关）、暗影史莱姆王（最终关），击败后城门碎裂
- **爽快手感**：不落地连续踩怪触发连击（分数 ×2/×4/×8）、打击停顿、过关烟花与三星结算动画
- **收集要素**：28 颗隐藏宝石、金币、红心
- **道具**：限时二段跳靴（30 秒）、吸铁石（20 秒金币宝石自动吸附）、无敌星（6 秒彩虹冲撞）、弹簧
- **机关**：移动平台、崩塌砖、冰面、尖刺、岩浆、炮台弹幕

## 操作

### 电脑端（键盘）

| 按键 | 功能 |
|------|------|
| ←→ / AD | 移动 |
| 空格 / W / ↑ | 跳跃（长按跳更高） |
| Esc / P | 暂停 |
| M | 静音 |
| R | 从检查点重来 |
| L | 暂停 / 结算界面返回选关地图 |

### 手机 / 平板（触屏）

- 打开网页即玩；竖屏时画面会自动旋转成横屏（系统锁定旋转、微信内置浏览器也没问题），把手机横过来握持即可
- 左下角 ◀ ▶ 虚拟键移动，右下角「跳」键跳跃（长按跳更高），支持多指同按与滑动切换
- 右上角按钮：全屏 / 声音 / 暂停
- 点击屏幕：选关地图点卡片开始、过关继续、阵亡后从检查点复活；暂停 / 结算界面左上角 ⌂ 返回选关
- 画面按设备像素比高清渲染，切后台自动暂停

音效与背景音乐由 WebAudio 实时合成，画面为 Canvas 手绘渲染，无任何外部资源。

## 本地运行

下载 `index.html`，双击用浏览器打开即可。
