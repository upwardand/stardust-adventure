# 星尘大冒险 Stardust Adventure

一个单文件 HTML5 横版平台闯关游戏——零依赖，打开就能玩。

**🎮 在线试玩：** https://upwardand.github.io/stardust-adventure/

## 玩法

- **10 个主题关卡**：绿茵草原 → 黄昏遗迹 → 熔岩堡垒 → 星空之巅 → 落日矿洞 → 云端天路 → 冰封峡谷 → 幽暗墓穴 → 风暴要塞 → 魔王王座
- **两场 Boss 战**：史莱姆王（第四关）与暗影史莱姆王（最终关）
- **收集要素**：22 颗隐藏宝石、金币、红心
- **道具**：限时二段跳靴（30 秒）、无敌星（6 秒彩虹冲撞）、弹簧
- **机关**：移动平台、崩塌砖、冰面、尖刺、岩浆、炮台弹幕

## 操作

| 按键 | 功能 |
|------|------|
| ←→ / AD | 移动 |
| 空格 / W / ↑ | 跳跃（长按跳更高） |
| Esc / P | 暂停 |
| M | 静音 |
| R | 从检查点重来 |

音效与背景音乐由 WebAudio 实时合成，画面为 Canvas 手绘渲染，无任何外部资源。

## 本地运行

下载 `index.html`，双击用浏览器打开即可。
